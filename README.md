# Java-Fullstack-Learning

## Overview
This repository documents my **60-Day Java Full Stack Learning Challenge**, where I consistently practice and upload **daily notes and code** related to Java and full-stack development.

The goal of this challenge is to build a **strong foundation in Core Java, Advanced Java, and Full Stack technologies**, while maintaining a structured and visible learning record.

---

## 60-Day Learning Challenge Goals
- Master **Core Java fundamentals**
- Learn **Object-Oriented Programming (OOP)**
- Understand **Collections, Exception Handling, Multithreading**
- Build backend applications using **JDBC, Servlet, JSP, Spring Boot**
- Work with **MySQL database**
- Learn frontend basics (**HTML, CSS, JavaScript**)
- Build and deploy **real-world projects**
